## 🚀 Kurulum

- Terminal'i aç ve aşağıdaki komutları çalıştır.

```
git clone https://github.com/heraklessii/Tsumi.git
cd tsumi
npm install
```

- Dosyaların inmesini bekle.
- `.env.example` dosyasını `.env` olarak değiştir ve doldur.
- `node .` yazarak botu başlat.
